# 第五次作业 Part 1
## 项目一：开发一个基于MCP协议的多 Agent 协作系统

### 输入输出定义
- 输入：用户问题（如“帮我写一篇关于AI Agent的文章”）
- 输出：多个 Agent 协作完成研究、撰写、审核、润色等任务

### 扩展项：
- 支持失败重试机制
